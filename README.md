# Gmail Integration

## Setup Instructions
1. Clone this repository:
   ```bash
   git clone https://github.com/jithendra-mallidu/Gmail-Integration.git
   ```
2. Navigate into the directory:
   ```bash
   cd Gmail-Integration
   ```

## API Key Configuration
1. Obtain your API key from the Google Developer Console.
2. Create a file named `.env` in the root directory and add your API key:
   ```env
   API_KEY=your_api_key_here
   ```

## Usage Guide
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the application:
   ```bash
   npm start
   ```
3. Follow the prompts to integrate your Gmail account with the application.
